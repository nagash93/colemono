HTML Structure - top

This theme is a fixed layout with two columns. All of the information within the main content area is nested within a div with an id of "content". The sidebar's (column #2) content is within a div with an id of "mainsidebar". The footer sidebar is within a div with an id of "footer". The general template structure is the same throughout the template.

CSS Files and Structure - top

I'm using two CSS files in this theme. The first one, style.css, contains a CSS reset, base typography, and the styles for the theme. This includes all 4 color schemes. The other CSS file, ie6.css, is used for correcting bugs specific to IE6.

JavaScript - top

This theme imports four Javascript files.

  1. jQuery
  2. scrollTo (a jQuery plugin)
  3. localScroll (a jQuery plugin that expands upon scrollTo)
  4. Custom scripts used for controlling animation, featured posts, and custom widgets.
  5. NOTE: IE6 will load DD_belatedPNG to correct PNG transparency.

PSD Files - top

The theme includes two PSD files. The main PSD file is properly layered to allow easy modification should you want to alter the theme.

If you'd like to change the logo in the header, open "header.psd", make the necessary adjustments, and then save the file as "logo.png" in the "images/logos/" folder. You may need to install the advent font (see Sources at the bottom of this documentation), or use a different font for your logo. The logo should be 56px tall.

Please note if you are using a color scheme other than the default (blue) the logo should be saved in the appropriate folder (e.g., "images/green/logos/").

Using Evolutionary Theme - top

Recommended Widget Setup - top

Widget Order

It is recommended that you place the Search widget first in your sidebar, followed by a group of ads (detailed below). Yes, widgets are intended to allow flexibility, but this will be the most aesthetically pleasing, particularly if using Featured Posts on the homepage.

Twitter Widget

You can use any twitter widget you like, but this theme is designed to work with the Wicket Twitter Widget or Twitter Tools. Using either widget will ensure your twitter widget is styled properly.

Custom Who's Talking? Widget

The Who's Talking widget is built into the Evolutionary Theme. This is a more advanced, modified take on the "Recent Comments" widget. It will display recent comments, as well as the most commented on posts from your blog. There are a few configurable options such as number to display, title, and order to display. These can be set when adding the widget to your sidebar.

Follow Us Widget with Icons

Evolutionary has styles that will automatically add rss feed and twitter icons to a sidebar widget if you wish. To do so, you will want to create a Text Widget, and title it Follow Us (or whatever you wish). Then place the following HTML in your text widget - replacing the URLs with your Twitter and RSS Feed URLs.

<ul>
<li><a href="http://timgweb.com/evolutionary/feed/rss/" class="rssfeed">RSS Feed</a></li>
<li><a href="http://twitter.com/timstl" class="twitter">Twitter</a></li>
</ul>

Evolutionary Theme Options Panel - top

This theme includes an options panel. To access, login to your admin panel, choose Appearance, and "Evolutionary Theme Options". This panel allows you to manage copyright text, theme color, and several options for customizing featured posts.

Setting Up Ads - top

Ad images should be 125x125 pixels. To add, simply open the Theme Options panel, fill out the Ad URL and path to your image on your server. The live demo of this theme uses ad images that are placed in the themes/evolutionary/images/ads/ folder. You can host your ad images wherever works best for you, though.

Featured Posts - top

This theme uses two jQuery plugins, along with custom jQuery code, to create a dynamic, animated "Featured Posts" section at the top of the homepage.

Making a Post Featured

First, if you haven't already, create a category called "Featured".

Next, write your post as normal, and associate it with your "Featured" category.

Then compose an "Excerpt" in the Excerpt box. This will be the text that shows up in the Featured Posts section, so it should be just a couple sentences.

Now you will need a graphic for your post. This graphic must be 650px wide by 290px tall. After you create this graphic, click the "Upload/Insert Image" button. Select the file on your computer, and upload. After the file uploads a screen should appear. If it does not, click the "show" link. Below the "Link URL" box is a button that says "File URL". Click that button, and copy the URL that appears in the "Link URL" box. Click Save all Changes, and close the Add an Image lightwindow.

Under the "Custom Fields" section we will add a custom field. Type the name "featuredpicture" (if this is your first) or select "featuredpicture" from the dropdown if one exists. In the "Value" box paste the "Link URL" you copied in the previous step. Click either "Add Custom Field" if this is your first Featured Post.

Lastly, Publish your post, and it will now appear in the Featured Posts section.

NOTE: If using a static homepage it's recommended you enable "Only show on frontpage" option in the admin.

Alternate Page Titles - top

You can add custom titles to your pages, so the text in the navigation, and heading text of your page will be different. To do so, just add a custom field called "alt_text" to your page. This text will display as the heading, while the standard page title will display in the navigation.

WordPress 3.0 Menus - top

Evolutionary 1.2+ supports WordPress 3.0's custom menus. This feature is available under Appearance -> Menus.

To activate, add a menu called "Main Nav" then add any pages, categories, or custom links that you wish to have on your menu.

Note: After activating the custom menu "Don't add Home page link to Nav?" admin setting will no longer be applicable, and the RSS icon will no longer appear. You can manually add a link to your RSS feed by adding http://domain.com/?feed=rss as a custom item, but it will not have the RSS icon.
