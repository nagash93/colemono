== Description ==

ML Post Slider  is a widget designed to display your slides in any widget area. 

[**DEMO**](http://www.mlipinski.pl/plugins/ml-post-slider)

You have to create special slider category which contains all slides which you want display in slideshow. Plugin use standard post editor to create slide. Widget have simple and short configuration. It use **JQuery** and **Cycle** plugin. This provides stable and professional working in any kind of web browser. More info in home page of plugin ([click!](http://www.mlipinski.pl/plugins/ml-post-slider/)).

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a special slides category.
4. Create slides which you want display in slider. How you do it is up to you. Add every slide to the category of slider.
5. Drop down widget to widget area which you want.
6. Configure widget, it is very simple.

More info about configuration [click!](http://www.mlipinski.pl/plugins/ml-post-slider/)
